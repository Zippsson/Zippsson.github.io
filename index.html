Feature project

Office Demons

The biggest project I have taken part in. (Spanning 6 months)
Office Demons is a four player couch co-op game with hectic turn-based combat with an emphasis on friendly fire and high jinks.
In this project I learned to work with AudioKinetic Wwise and I also worked a lot with Unity's UI & Eventsystem tools, I also designed most of the menues as well as some of the 3d models. This is also the project in which I helped implement a PCG system, that helped us generated levels and object placements automatically.

Other projects

Here you can see a selection of other projects that I have taken part in.

Disco Cube Puzzler

The first big project that I took part in (Spanning 20 weeks), in which I took on a leadership role. I also experimented a lot with movement around other 3D objects using edge-triggers in order to rotate the main level cube.

Kessler Syndrome

My groups contribution to 2021 GlobalGameJam. Where I created a "random spawning" algorithm that creates randomly chosen items from a list, gave them a value and a random direction. This was created using a modified BOID algorithm. The theme for that years Game Jam was "Lost and Found".

MagnetMagnus

My groups contribution to 2022 GlobalGameJam. In which I wrote, recorded and implemented all the music and SFX in the game.
